# 🚀 Bóveda de Notas Automatizada

**Un Entorno de Conocimiento Visual y Context-Aware impulsado por automatización.**

A diferencia de los sistemas tradicionales de apuntes estáticos, esta bóveda fusiona el **pensamiento visual orgánico** (diagramas interactivos, imágenes y tableros) con una **arquitectura de enlaces rígida gestionada por código**.

Está diseñada bajo la filosofía de "fricción cero": el usuario solo debe preocuparse por aprender y documentar, mientras que el sistema, a través de scripts en JavaScript y la API de Obsidian, se encarga de la infraestructura subyacente, la categorización y la generación de espacios de trabajo.

---

## 💡 ¿Por qué usar este sistema?

El problema con la metodología Zettelkasten tradicional es la sobrecarga cognitiva: decidir dónde guardar una nota, cómo enlazarla y qué estructura darle interrumpe el flujo de estudio o desarrollo. Esta bóveda resuelve eso mediante un enfoque **Context-Aware** (consciente del contexto).

- 🧠 **Flujo Visual Integrado:** No son solo notas de texto. El sistema despliega automáticamente pizarras de Excalidraw y tableros Kanban listos para usar según lo que estés creando.
    
- 🤖 **Enlazado Inteligente:** Los scripts detectan el estado actual de tu interfaz, tu historial de navegación y tus metadatos para heredar etiquetas y crear vínculos automáticamente sin intervención manual.
    
- 🏗️ **Infraestructura Dinámica:** Funciona como un inventario tecnológico. El sistema lee tu bóveda en tiempo real para ofrecerte opciones dinámicas (como armar el stack tecnológico de un proyecto basándose en tus notas de software existentes).
    

---

## ⚙️ ¿Qué hace cada plantilla por ti?

El corazón del sistema son sus plantillas automatizadas. Al crear una nota nueva, no te enfrentas a una página en blanco. El sistema te hace un par de preguntas rápidas y **construye un espacio de trabajo completo** adaptado a lo que necesitas hacer:

- **🚀 Proyecto (Desarrollo y Gestión):** Prepara todo para que empieces a codificar. Te genera un tablero Kanban para tus tareas, un lienzo infinito para diagramar la arquitectura, y te permite seleccionar tu _Stack Tecnológico_ de un menú interactivo.
    
- **💻 Software (Inventario Tecnológico):** Tu propio diccionario técnico. Documenta lenguajes, frameworks o herramientas. El sistema usará estas notas más adelante para sugerírtelas automáticamente al armar tus proyectos.
    
- **🎓 Tema (Estudio Teórico):** El punto de partida de tus estudios. Asigna automáticamente la materia que estás cursando y te despliega una pizarra visual para que armes mapas mentales de la teoría.
    
- **✏️ Práctico (Laboratorios y Guías):** Para resolver tareas complejas. Te crea un espacio de resolución que **se conecta por sí solo** con tus notas de teoría.
	
- **🧠 Concepto (Captura Atómica):** El bloque fundamental de tu conocimiento. Ideal para definiciones puras, patrones de arquitectura o teoremas. El sistema rastrea de que nota se origina y hereda la tag que da el contexto automáticamente.
	
- **📝 Ejercicio (Resolución Rápida):** Notas atómicas para problemas matemáticos o algoritmos específicos. El sistema es tan inteligente que sabe desde qué tema o práctico vienes y etiqueta la nota por ti.
    
- **📔 Bitácora (Trazabilidad):** Tu diario de desarrollo. Registra en qué trabajaste hoy y qué bugs quedaron pendientes. El sistema detecta tu proyecto activo y lo vincula todo para que nunca pierdas el hilo de tu avance.

## 📦 Instalación Rápida

1. **Generar tu Repositorio:** Utiliza el botón _"Use this template"_ en GitHub para clonar este entorno con un historial limpio.
    
2. **Despliegue:** Abre la carpeta clonada como una nueva bóveda (Vault) desde la aplicación Obsidian.
    
3. **Confianza en el Código:** Al ser un sistema que ejecuta scripts de automatización, Obsidian te pedirá confirmación. Selecciona **"Trust author and turn off Safe Mode"** para permitir que el motor procese la lógica.
